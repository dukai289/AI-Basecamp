# 知识库

:::tip[note] 
这里沉淀相对长期有效的 AI 内容，重点是基础知识、可复用经验、工具框架、教程方法和资源索引。
:::


## 内容结构

| 板块 | 内容 |
| --- | --- |
| 知识体系 | AI 基础知识、概念解释、论文笔记和体系化资料 |
| 教程与指南 | 实操教程、上手指南、配置流程和案例步骤 |
| 模型评测 | 模型体验、能力评测、对比结论和使用建议 |
| 工具与框架 | Agent 开发、微调、测评、部署等工程实践 |
| 方法论 | 工作流、Prompt 策略和最佳实践 |
| 资源 | 课程、书籍、论文集合、GitHub 项目、数据集和工具站 |

## 维护原则

- 稳定知识优先放入知识库，短期新闻和产业动态放入资讯。
- 评测类内容记录模型版本、任务样例、环境和限制条件。
- 教程类内容写清楚前置条件、步骤、命令和预期结果。
- 工具类内容记录定位、适用场景、使用示例和常见问题。
- 方法论内容优先沉淀可复用流程，而不是一次性经验。

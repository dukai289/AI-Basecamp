# 知识库

:::tip[note]
这里沉淀相对长期有效的 AI 内容，重点是基础知识、可复用经验、工具框架、教程方法和资源索引。
:::

## 如何使用知识库

| 你想解决的问题 | 推荐入口 | 内容 |
| --- | --- | --- |
| 这个概念是什么？原理是什么？ | 知识体系 | AI 基础知识、概念解释、论文笔记和体系化资料 |
| 这个东西怎么做？ | 教程与指南 | 实操教程、上手指南、配置流程和案例步骤 |
| 这个工具适合什么场景？ | 工具与框架 | Agent 开发、微调、测评、部署等工程实践 |
| 这个模型表现如何？ | 模型评测 | 模型体验、能力评测、对比结论和使用建议 |
| 这类事情怎么稳定做好？ | 方法论 | 工作流、Prompt 策略和最佳实践 |
| 从哪里学习更多？ | 资源 | 课程、书籍、论文集合、GitHub 项目、数据集和工具站 |

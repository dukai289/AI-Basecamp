# AI-Basecamp


:::tip[note]
一个面向 AI 学习、实践与协作的开放知识基地。  
聚合值得长期关注的产业动态、知识、工程实践与方法论，和更多同行一起交流、验证与进步。
:::


## 内容导航

| 序号 | 板块 | 内容 |
| :---: | :---: | :---: |
| 1 | 动态 | 站点内容、结构和配置变更 |
| 2 | 资讯 | AI 行业动态、产品发布、重要论文、产业政策和事件跟踪 |
| 3 | 知识库 | AI 基础知识、教程指南、模型评测、工具框架、方法论和资源清单 |

## 知识库结构

| 序号 | 板块 | 内容 |
| :---: | :---: | :---: |
| 1 | 知识体系 | 基础知识、概念解释、论文笔记和体系化资料 |
| 2 | 教程与指南 | 实操教程、上手指南、配置流程和案例步骤 |
| 3 | 工具与框架 | Agent 开发、微调、测评、部署等工程实践 |
| 4 | 模型评测 | 模型体验、能力评测、对比结论和使用建议 |
| 5 | 方法论 | 工作流、Prompt 策略和实践心得 |
| 6 | 资源 | 课程、书籍、论文、项目、数据集和工具站 |

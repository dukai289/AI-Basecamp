# AI-Basecamp

:::tip[note]
一个面向 AI 学习、实践与协作的开放知识基地。  
聚合值得长期关注的产业动态、知识、工程实践与方法论，和更多同行一起交流、验证与进步。
:::

## 你可以在这里找到什么

| 入口 | 适合场景 | 内容 |
| :---: | :---: | --- |
| changelog | 想了解站点最近更新 | 内容、结构变更 |
| 资讯 | 想跟踪近期 AI 行业变化 | 产品发布、模型动态、论文、政策、产业事件 |
| 知识库 | 想系统学习或复用经验 | 基础知识、教程、评测、工具框架、方法论、资源 |

## 推荐阅读路径

- LLM 入门：先看 [LLM 学习路线图](./knowledge/llm-basics/LLM学习路线图.md) 和 "知识体系 / LLM 基础" 等
- 模型部署：先看显存估算、GPU、vLLM 等
- Agent 实践：先看 Agent 开发、Agent 实例等
- 信息跟踪：先看每日 AI 动态，如果需要获取更多资讯可以参考 [资讯与科普源](./resources/news.md)

## 内容原则

- 资讯追求及时
- 知识库追求长期可复用
- 教程追求可复现

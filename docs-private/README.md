# 私人笔记

这个目录用于存放需要 Git 同步、但不作为站点内容发布的私人笔记。

约定：

- `docs/`：脱敏后的发布版笔记，会进入 Docusaurus 站点。
- `notes-private/`：原始私人笔记，可以被 Git 跟踪，但不在 Docusaurus 侧边栏、博客或静态资源里引用。
- 远程仓库要保持私有。
- 从这里整理文章到 `docs/` 前，先复制一份再脱敏，不要直接移动原始笔记。

---

步骤：

1. 新笔记先放到 `notes-private/`。
2. 需要发布时，从 `notes-private/` 复制一份到 `docs/` 对应分类目录。
3. 在 `docs/` 版本里脱敏：域名、IP、账号、Token、Cookie、日志样例、客户/公司信息、内部路径都替换掉,确认没有真实敏感信息。
4. `notes-private/` 可以 Git 同步，同时。

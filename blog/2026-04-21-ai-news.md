## 头条要闻

### Anthropic × Amazon：算力合作升级到 5GW，云采购承诺超 1000 亿美元

4 月 20 日，Anthropic 宣布与 Amazon 深化合作，新的协议将为 Claude 训练与部署锁定最多 5GW 算力。Anthropic 表示，未来十年将向 AWS 技术投入超过 1000 亿美元，覆盖 Graviton 与 Trainium2 到 Trainium4 芯片，并继续把 AWS 作为关键训练与云基础设施提供方。

这件事的重要性不只是“又一笔大单”，而是前沿模型公司的核心竞争点继续向“长期可锁定算力”倾斜。过去大家更关注模型榜单、参数量和 API 价格，现在越来越多的大动作发生在芯片、集群、区域推理部署和长期采购协议上。对于 Claude 来说，这意味着它在 Bedrock 上的企业分发和在 AWS 上的训练基础会进一步绑定。

中文媒体跟进显示，Amazon 还将立即追加 50 亿美元投资，并在满足里程碑后追加至多 200 亿美元。这使合作结构同时覆盖资本、芯片与云消费三层，已经不是普通意义上的云厂商供货关系。

[citation:Anthropic](https://www.anthropic.com/news/anthropic-amazon-compute) · [citation:IT之家](https://www.ithome.com/0/941/387.htm)

### Microsoft 收购 Fintool：AI 金融研究工具走向 Office 工作流

4 月 19 日，Fintool 创始团队确认公司已被 Microsoft 收购。Fintool 的定位是面向金融专业人士的 AI 研究工具，能够读取财报电话会、分析公司文件、综合研究材料，并在新版本中进一步自动生成 Excel 中的 DCF 模型、PowerPoint 业绩演示文稿和 Word 研究备忘录。

这笔交易的意义在于，AI 正继续从“聊天入口”往高价值垂直工作流里下沉。金融研究是一个高信息密度、高文档密度、高模板化交付的场景，天然适合 Agent 化。若 Microsoft 进一步把这类能力接进 Microsoft 365，Office 套件可能不只是调用模型，而是直接承接一整类专业分析任务。

## 模型发布与更新

### Google AI Studio：AI Pro / Ultra 订阅开始直连开发者原型链路

Google 最新更新显示，从今天开始，Google AI Pro 和 Ultra 订阅用户在 Google AI Studio 中获得更高使用额度，并可直接访问 `Nano Banana Pro` 与 `Gemini Pro` 模型。Google 同时强调，这让订阅用户可以更快从原型尝试过渡到实际应用开发，而不必一开始就进入标准 API 计费模式。

这个变化不算“基础模型发布”，但对开发者生态很关键。它把消费订阅、实验性原型和开发者工具串成了一条更短的路径。过去很多平台把“聊天产品”和“开发平台”切得很开；Google 这次则是在尝试把订阅用户直接转化为轻量开发者和原型构建者。

## 融资与交易

### Amazon / Anthropic：投资、采购与芯片协同三线绑定

如果只看表面，这是一条“追加投资”的新闻；但更值得注意的是交易结构。中文媒体披露，Amazon 将立即追加 50 亿美元投资，后续还有最多 200 亿美元与商业里程碑绑定；与此同时，Anthropic 承诺未来十年在 AWS 技术上投入超过 1000 亿美元。

这种结构意味着 AI 大厂与云厂商的关系正在从“融资 + 上云”进一步升级为“资本绑定 + 芯片协同 + 长期算力消耗承诺”。对行业来说，这会继续抬高前沿模型竞争的门槛，也会让自研芯片和云生态绑定变得更重要。

### Microsoft / Fintool：垂直 Agent 并购仍在继续

Fintool 不是通用聊天产品，而是垂直金融研究 Agent。它被收购说明，大模型行业的并购重点正在从“再买一个模型团队”转向“直接买下已验证的垂直工作流”。

这类交易的看点在于落地位置很清楚：财报、估值模型、演示文稿、研究备忘录，都是金融行业每天都在反复生产的标准化产物。和单纯的问答助手相比，这更容易直接转化为企业采购预算。

## 全球产业动态

### 北京人形机器人半马：具身智能开始进入公开、高强度验证场景

4 月 19 日举行的 2026 北京亦庄人形机器人半程马拉松，成了过去 48 小时里最具传播力的具身智能事件之一。36 氪和英文媒体都提到，赛事规模已经扩展到 26 个品牌、超百支队伍、300 余台人形机器人。冠军“闪电”机器人成绩大幅刷新去年的纪录，英文媒体还特别关注到其成绩已经快于当前男子半马世界纪录。

这场比赛的重要意义不是“机器人会跑步”本身，而是它把具身智能从实验室演示推进到了公开、长距离、连续性、可围观的真实环境测试。耐力、稳定性、自主导航、散热、能耗、地形适应和故障恢复，这些都是未来机器人进入工厂、仓储、巡检和公共空间前必须跨过的门槛。

如果说 2025 年的同类赛事更像展示秀，那么 2026 年这次比赛更像一次行业压力测试。对中国机器人产业来说，这也是“硬件量产 + 具身算法 + 公共赛事验证”三者合流的典型信号。

## 发布判断

- [ ] 内容适合公开发布，不包含未授权信息
- [ ] 内容已经从私人语境改写为公开文档语境
- [ ] 技术结论、命令、配置和链接已复核
- [ ] 引用外部资料时保留来源链接
- [ ] 不确定的信息已删除、泛化或标注为待确认

## 敏感信息检查

### 身份与联系方式

- [ ] 删除或泛化真实姓名、昵称、工号、组织架构信息
- [ ] 删除邮箱、手机号、微信号、飞书 ID、GitHub 私有账号等联系方式
- [ ] 删除客户、供应商、合作方、内部团队等未公开主体名称

### 凭证与访问控制

- [ ] 删除 token、API key、secret、password、cookie、session、SSH key
- [ ] 删除临时验证码、授权链接、签名 URL、对象存储私有链接
- [ ] 删除 `.env`、私有配置、认证头、数据库连接串
- [ ] 删除可直接复现登录或提权路径的细节

### 网络与基础设施

- [ ] 删除或泛化公网 IP、内网 IP、域名、端口、主机名
- [ ] 删除服务器路径、用户名、主机拓扑、Nginx 私有配置
- [ ] 删除安全组、防火墙、白名单、跳板机、VPN 等访问细节
- [ ] 删除日志中的 trace id、request id、用户 id、设备 id

### 业务与数据

- [ ] 删除真实业务数据、订单、合同、金额、成本、收入、报价
- [ ] 删除客户案例中的可识别细节
- [ ] 删除内部路线图、未发布功能、未公开模型或实验结果
- [ ] 删除原始用户输入、聊天记录、会议发言和个人评价

### 图片、附件与代码

- [ ] 截图已打码账号、头像、浏览器地址栏、终端路径和通知信息
- [ ] 附件已确认不包含隐藏 sheet、批注、修订记录和元数据
- [ ] 示例代码使用占位符，不包含真实密钥、路径和私有包地址
- [ ] 命令输出已清理用户名、机器名、目录、日志和环境变量

## 推荐替换方式

| 类型 | 不建议公开 | 推荐写法 |
| --- | --- | --- |
| IP 地址 | `10.12.34.56` | `<internal-ip>` |
| 域名 | `private.example.com` | `<internal-domain>` |
| 用户名 | `zhangsan` | `<user>` |
| 路径 | `/home/zhangsan/project` | `/path/to/project` |
| Token | `sk-xxxx` | `<api-key>` |
| 数据库 | `mysql://user:pass@host/db` | `<database-url>` |
| 客户名 | `某真实客户` | `某客户` / `示例客户` |
| 金额 | `¥123456` | `约 xx 万` / 删除 |
